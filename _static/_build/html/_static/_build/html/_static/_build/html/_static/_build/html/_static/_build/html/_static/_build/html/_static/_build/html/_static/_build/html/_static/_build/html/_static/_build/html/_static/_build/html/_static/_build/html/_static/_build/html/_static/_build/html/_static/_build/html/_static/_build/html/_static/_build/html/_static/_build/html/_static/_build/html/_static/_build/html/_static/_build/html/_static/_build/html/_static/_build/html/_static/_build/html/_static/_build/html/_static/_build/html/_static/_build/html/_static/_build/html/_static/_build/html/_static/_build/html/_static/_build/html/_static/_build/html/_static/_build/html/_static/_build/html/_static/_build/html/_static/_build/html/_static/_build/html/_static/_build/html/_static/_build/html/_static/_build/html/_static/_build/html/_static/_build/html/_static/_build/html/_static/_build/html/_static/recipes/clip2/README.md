# The Second Cadenza Lyric Intelligibility Challenge

Cadenza challenge code for the second Cadenza Lyric Intelligibility Challenge (CLIP2)

For more information about CLIP2, please [visit the Cadenza Challenge website](https://cadenzachallenge.org/).

The directory contains [baseline code](baseline/). To run it, install the baseline extras (e.g. `pip install -r baseline/extra_requirements.txt`) and execute the scripts from within the `baseline/` folder.
