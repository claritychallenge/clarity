PyClarity documentation
=======================

PyClarity provides the core tools and functionalities used across
`Clarity Challenges <https://claritychallenge.org/>`_ and `Cadenza Challenges <https://cadenzachallenge.org/>`_.

Clarity Challenges
------------------

* `3rd Clarity Prediction Challenge (CPC3) <https://claritychallenge.org/docs/cpc3/cpc3_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cpc3>`__)
* `3rd Clarity Enhancement Challenge (CEC3) <https://claritychallenge.org/docs/cec3/cec3_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cec3>`__)
* `2nd Clarity Prediction Challenge (CPC2) <https://claritychallenge.org/docs/cpc2/cpc2_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cpc2>`__)
* `The ICASSP 2023 Clarity Challenge <https://claritychallenge.org/docs/icassp2023/icassp2023_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/icassp_2023>`__)
* `2nd Clarity Enhancement Challenge (CEC2) <https://claritychallenge.org/docs/cec2/cec2_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cec2>`__)
* `1st Clarity Prediction Challenge (CPC1) <https://claritychallenge.org/docs/cpc1/cpc1_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cpc1>`__)
* `1st Clarity Enhancement Challenge (CEC1) <https://claritychallenge.org/docs/cec1/cec1_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cec1>`__)


Cadenza Challenges
------------------

* `2nd Cadenza Challenge (CAD2) <https://cadenzachallenge.org/docs/cadenza2/intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cad2>`__)
* `The ICASSP 2024 Cadenza Grand Challenge <https://cadenzachallenge.org/docs/icassp_2024/intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cad_icassp_2024>`__)
* `1st Cadenza Challenge (CAD1) <https://cadenzachallenge.org/docs/cadenza1/cc1_intro>`_ (`GitHub Recipes <https://github.com/claritychallenge/clarity/tree/main/recipes/cad1>`__)
